# 🎬 와인 소믈리에 추천 서비스 데모 영상 시나리오

본 시나리오는 **오픈소스소프트웨어실습 기말 대체 과제** 데모 영상 촬영을 위해 작성되었습니다. 평가 기준의 모든 필수 요소(EC2 접속, docker ps, 실시간 API 통신 증적 등)를 포함하고 있습니다.

* **예상 시간**: 약 3분 30초 ~ 4분
* **녹화 전 준비사항**:
  1. AWS EC2 인스턴스 구동 및 도커 서비스 실행 (`docker-compose up -d`)
  2. EC2 터미널 SSH 연결창 활성화
  3. 웹 브라우저 준비

### 💡 촬영 팁 & 연출 조언
1. **화면 분할 캡처**: 브라우저창(좌측)과 터미널창(우측)을 동시에 녹화화면에 띄워두고 진행하면 흐름 끊김 없이 통신을 검증하기에 매우 용이합니다.
2. **배포 주소 확인**: 브라우저 창에서 AWS EC2 퍼블릭 IP와 포트 번호(`:8501`)가 선명하게 노출되도록 캡처해 주세요.
3. **단호하고 신뢰감 있는 목소리**: L1 거리(맨해튼 거리) 오차 연산이나 음식 가중치 설정 등 백엔드 알고리즘의 동작 원리를 언급할 때 차분하고 신뢰감 있는 목소리 톤을 유지하면 발표 점수에 긍정적입니다.
